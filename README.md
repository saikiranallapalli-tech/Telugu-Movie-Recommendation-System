# Telugu Movie Recommendation System

## Overview

A Java-based Telugu Movie Recommendation System that recommends similar movies based on genre similarity.

## Features

* Genre-based movie recommendations
* Telugu movie dataset
* Similarity score calculation
* Console-based application

## Technologies Used

* Java
* OOP Concepts
* ArrayList
* HashMap

## Sample Output

Input:
Pushpa

Output:
Baahubali
RRR
Magadheera
HIT
Kalki 2898 AD
